# AI-Tool-to-Read-and-Analyze-Legal-Contracts-Automatically_Infosys_Internship_Jan26
AI-powered system designed to automate the process of contract analysis, improving efficiency and precision while generating customized, actionable reports. It leverages a multi-agent framework where each AI agent specializes in a distinct domain such as compliance, finance, and operations to deliver comprehensive and professional insights.


All the folders/files are the documentation of the work done in the internship(2 months)

To run the main app, only extract "Final" folder and run the uvicorn command (uvicorn app:app --reload) and access the frontend on localhost 8000 or **http://127.0.0.1:8000/**
