# Screenshots and Videos

## How to Add Media Files

1. **Take Screenshots:**
   - Upload page: `docs/images/upload.png`
   - Processing: `docs/images/processing.png`
   - Results: `docs/images/results.png`
   - Analysis detail: `docs/images/analysis-detail.png`

2. **Record Demo Video:**
   - Full workflow demo: `docs/videos/demo.mp4` or `demo.gif`

3. **Add to Git:**
   ```bash
   git add docs/
   git commit -m "docs: Add screenshots and demo video"
   git push
   ```

## Recommended Tools

- **Screenshots:** macOS Screenshot (Cmd+Shift+4), Windows Snipping Tool
- **Video:** QuickTime (macOS), OBS Studio, or convert to GIF using ezgif.com
- **GIF Size:** Keep under 10MB for GitHub
